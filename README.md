# iedora-app

The iedora application monorepo (Bun workspaces): the Next.js 16 web app (`apps/web`, serving menu.iedora.com — multi-tenant menu builder + auth — and iedora.com landing over a host-based rewrite) together with the Bun/Hono backend services (`services/`) that own data, auth and business rules. Built and deployed via Kamal (see `iedora/infra`).

📖 Documentation: https://docs.733113.xyz
